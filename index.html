My Journal

Write it down.
Keep it yours.

Research consistently shows that regular journaling reduces stress, sharpens clarity, and builds resilience over time. MindEase makes the habit effortless.

  • ⏱
    Live Date & Time A live clock updates every second. Your entry is stamped with the exact moment automatically — no manual dating needed.
  • ✏️
    Guided Gratitude Prompts Four research-backed prompts are always visible. Use them as inspiration or ignore them — write whatever is on your mind.
  • 📁
    Saved to Your Own Files Entries save as .txt files in ~/Documents/MindEase_Journal/ — readable in any app, forever.
  • 📖
    Browse Past Entries All saved journals listed newest-first in a side panel. Click any entry to read its full content in a clean, distraction-free preview.

Design Principles

Why it's built
this way.

01 — Principle
Name the stressor first

Before showing resources, the app asks what you're stressed about. Naming it reduces cognitive load and makes the experience feel personal rather than encyclopedic.

02 — Principle
Local-first, privacy-first

A wellness app for students discussing personal struggles should never upload data to a server. Journal entries stay on your computer — always accessible, never shared.

03 — Principle
Real information only

Every resource card shows the actual room number, phone number, and email address — not a general "visit the website" prompt. Friction kills follow-through.

Technical

Built with
modern C++.

Every feature is a real C++ or Qt concept in action — not a textbook example, but a working, polished application students actually use.

"Clean code is not written by following rules. It is written by someone who cares."

C++17 Qt 6 Widgets qmake QSS Theming QFile · QTextStream QStandardPaths QDateTime QDesktopServices QTimer · QMap
OOP / Classes All screens are QWidget subclasses; MainWindow inherits QMainWindow
Lambdas Signal connections use [this, id]() captures throughout all modules
Signals & Slots Qt event system wires navigation, accordion toggles, journal saves, and clock ticks
File I/O QFile + QTextStream for persistent read/write of journal entries
Qt Containers QMap, QList, QVector, QStringList across all modules
Dynamic UI Resource cards, topic cards, and accordion folders built at runtime from static data
C++17 Structs ResourceInfo, FolderDef, ToolItem with brace initialization
